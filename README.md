# BEAUTE_INEE

Site web de Beauté Inée.

## Structure

- Fichiers PHP principaux
- Dossier `css` pour les styles
- Dossier `js` pour les scripts
- Dossier `images` pour les médias
- Dossier `php` pour les traitements côté serveur

## Installation

1. Copier le projet dans un serveur PHP.
2. Vérifier la configuration de `config.php`.
3. Ouvrir le site dans un navigateur.

## Notes

- Ajouter un dépôt distant GitHub avec `git remote add origin <url>`.
- Pousser avec `git push -u origin main`.
